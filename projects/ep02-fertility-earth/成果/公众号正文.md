# Vibe Analysis EP02 | 东亚不是特例，它只是先掉下去了

前几天，我听了一期罗永浩和梁建章关于人口问题的对谈节目。

一个人从长期趋势和普通人感知出发，一个人从人口经济学和国家竞争力出发。

听完之后，我脑子里卡住了一个问题：

**东亚到底是全球人口问题的例外，还是只是最先把问题演出来的地方？**

如果按以前的习惯，我大概率会先去看几篇文章，再记几个数字，然后得到一个模糊但正确的判断：韩国很低，日本很低，中国也很低，东亚挺危险。

但这次我不想停在“知道几个数字”这一步。

我想把这件事看完整。

于是我做了两件事：

第一，让我的 `Hermes Agent` 去收集和整理数据；
第二，用我自己的 `Vibe Analysis` 技能链，把这个问题做成一张真正能讲故事的世界地图。

结果比我预期快得多。

大概 2 个小时，Hermes Agent 把国家级人口、生育率、城市化和区域历史资料都整理出来了。接着我用 Vibe Analysis 的流程，把它做成了一张可拖动时间轴、可点国家、可看城市穿透的交互地图，再顺手把文章主线也搭了出来。

老实说，这次最让我兴奋的，不只是效率。

而是它最后给出的洞察深度，**明显超过了我原本对“快速分析”这件事的预期**。

它不是简单把数据画出来。

它真的把一个本来很容易落入情绪化讨论的话题，重新拉回到了结构层面。

## 第一部分：成果展示

### 一句话总结

我用 Hermes Agent + Vibe Analysis，在很短时间内完成了一次全球人口主题的可视化分析：

- 整理国家级人口、生育率、城市化与区域历史数据
- 构建 1960—2023 年全球总和生育率交互地图
- 输出一张适合录屏传播的 HTML 叙事地图

最后得到的核心判断是：

**东亚不是低生育的例外，它只是最先掉下去、也掉得最深的那一块。**

### 发现 1：最吓人的不是低，而是回不去

很多人看到韩国 0.72，会本能地觉得，最危险的是“有国家跌得特别低”。

但我做完地图以后，真正让我发凉的，不是最低值本身。

而是另一件事：

**跌破 1.5 之后，几乎没有国家真正回到 2.1 的人口更替水平。**

我专门把那些曾经跌破 1.5、后来又有所回升的国家单独标了出来。

比如：

- 捷克：1.14 → 1.70
- 瑞典：1.54 → 1.70
- 爱沙尼亚：1.36 → 1.60

看起来像在恢复，对吧？

但问题是，它们都没有真正回去。

这意味着 1.5 很可能不是一个普通阈值，而是一条单行道的入口。

回升可以发生。

逆转几乎没发生过。

这也是我现在对很多生育政策讨论最大的保留：

我们总是下意识地把“回升”理解成“终将回到正常”，但从全球数据看，现实远没有这么乐观。

---

### 发现 2：城市化比文化更冷酷

中文互联网谈低生育，最喜欢归因的几个词是：东亚文化、儒家传统、鸡娃、房价、内卷。

这些都不是错。

但如果把视角拉到全球，你会发现有一个变量，比这些词更稳定，也更冷酷。

那就是城市化。

地图上最直观的画面，是那些又大、又红的点，几乎扎堆出现在高度城市化地区。

- 韩国：城市化 82%，TFR 0.72
- 日本：城市化 92%，TFR 1.26
- 大量发达国家也都落在同一片低生育颜色里

真正有意思的是例外。

比如以色列和蒙古。

它们提醒我们：文化、宗教、社会结构当然仍然有作用，但在大多数国家层面，**城市化率才是低生育最强的背景变量之一**。

换句话说，问题不只是“年轻人想不想生”。

而是当一个社会从农村结构走向城市结构，从大家庭走向核心家庭，从低成本生活走向高流动、高教育、高居住成本生活时，生育这件事会被整套重新定价。

这套定价机制，比单纯的文化解释更硬，也更难被短期政策逆转。

---

### 发现 3：韩国真正可怕的，不是 0.72，而是 0.72 来得太快

低生育国家并不是韩国独有。

德国有过，日本有过，俄罗斯也有过。

韩国真正让人不安的，不只是它现在最低。

而是它跌到这个位置的速度，已经超出了工业化国家的常见历史经验。

也就是说，它不是普通地下滑。

它是加速下滑。

很多人看 0.72，会把它当成一个静态数字。

但地图真正厉害的地方在于，它会把这个数字重新变回一段历史速度。

你拖着时间轴往前走，会发现它不是“突然变低”。

它是一个社会在非常短的时间里，把自己推到了一个极端位置。

这会带来什么？

- 劳动力收缩
- 代际扶养比恶化
- 教育、住房、就业和财政体系被反向重写

所以韩国不是一条“极端新闻”。

它更像一个样本国家，提前把很多社会未来要面对的问题演了一遍。

---

### 发现 4：中国真正难讲的，不是 1.09，而是 1.09 后面那道代际断层

中国的人口问题，也常常被一个数字讲窄了。

1.09。

这个数字当然低，也当然重要。

但如果你只盯着 1.09，看见的只是一个全国平均值。

而真正复杂的地方，在它背后那条代际断层。

一旦把全国数字、超大城市、代际差异放在一起看，你会发现：

中国的问题不是“所有人一起往下掉”。

而是不同代际正在以不同速度掉下去，最后由平均数把这些差异抹平。

上海、北京这种超大城市的极低生育，很多人已经听麻了。

但真正值得警惕的，不是“有几个地方特别低”。

而是这种状态，正在从局部样本变成一整代人的现实背景。

这意味着什么？

意味着中国的问题，不能只用一句“年轻人不想生”解释。

那句话太轻了。

它像在描述一种偏好。

但人口结构变化从来不是偏好问题那么简单。

它是一列已经启动的慢车。

等你意识到它往哪开的时候，很多车厢其实已经出站了。

AI 真正厉害的，不是单点替代某个分析动作，而是把原本分散在不同工具、不同角色、不同心智模式里的环节，压成一条连贯工作流。

## 第二部分：方法论

### 这次为什么能做得这么快

如果按传统方式做这类主题，你通常会经历下面几个阶段：

1. 找资料
2. 看资料
3. 自己记笔记
4. 自己做图
5. 再想怎么写

每一步都不算特别难，但加起来非常碎。

而我这次做法刚好相反：

先把问题问清楚，再让 Agent 和 Skill 链条往下跑。

这次真正的起点，不是我打开 Excel，也不是我开始写图代码。

而是我先把问题压成了一句清楚的话：

> 东亚到底是人口问题的特例，还是全球趋势中最早、最深的那一段？

有了这个问题，后面的事情就能被分工：

- `Hermes Agent` 负责收集、整理、归档资料
- `Vibe Analysis` 负责把问题转成分析任务，再把结果变成可传播输出

### 我这次大概用了什么流程

#### 1）先让 Hermes Agent 收集数据

这一步我没有自己手动到处翻表。

我让 Hermes Agent 去整理：

- 国家级人口与生育率资料
- 城市化与城市/区域历史数据
- 数据来源说明与审计说明

大概 2 个小时左右，基础资料就已经能用了。

这一步给我的最大感受是：

**以前做分析最容易卡在“资料还没准备好”，现在这个环节已经被大幅压缩了。**

#### 2）再用 Vibe Analysis 把问题变成产物

有了数据之后，真正关键的不是“怎么画图”，而是“怎么讲这个问题”。

这也是我一直觉得 Vibe Analysis 比单纯 Vibe Coding 更有意思的地方。

它不是只帮你出一个产物。

它会逼你先想清楚：

- 你到底想回答什么问题
- 哪个变量是主叙事
- 什么发现值得讲
- 什么表达适合传播

这次我在 EP01 的基础上，把技能链继续往前推了一步：

- `vibe-analysis-taskwriting`：把问题压成任务书
- `vibe-analysis-world-map`：构建世界地图和国家级叙事编码
- `warm-minimal-dashboard`：保证输出适合截图和传播

最后出来的不是单个图，而是：

- `dashboard.html`
- `insights.md`
- `report.md`
- 公众号正文

这也是为什么我会说，这次的结果远超我的预期。

不是因为它特别复杂。

恰恰是因为它在很短时间里，把“效率高”和“洞察深”这两个通常互相冲突的东西，同时做到了。
